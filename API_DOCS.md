# API_DOCS

## OpenAPI / Postman
- Generate OpenAPI (Swagger) from Spring controllers using `springdoc-openapi`.
- Export Postman collection from Swagger UI for quick testing 

### Example to export:
1. Start server (`dev` profile).
2. Open `/swagger-ui/index.html`.
3. Use UI to generate/export OpenAPI JSON.
4. Import to Postman or CI tests.
